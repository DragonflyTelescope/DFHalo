# DFHalo
Halo Assessment for single exposures taken by Dragonfly

### Dependencies

- matplotlib
- numpy
- scipy>=1.6.0
- astropy>=4.0
- photutils>=0.7.2
- scikit-learn>=0.24.2
- tqdm

- openjdk

```bash
conda install -c conda-forge openjdk
```

### Installation
---

```bash
cd <install directory>
git clone https://github.com/DragonflyTelescope/DFHalo.git
cd DFHalo
pip install -e .
```
